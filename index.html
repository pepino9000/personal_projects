CRUD inventario


CRUD

El inventario de productos permite realizar las 4 funciones básicas de una base de datos: crear, leer, actualizar y borrar.

Servidor http

Utilizando javascript se ha implementado un servidor http, el cual a través de url y métodos ejecuta las funciones del CRUD.

BDD postgreSQL

Para este programa se ha utilizado el motor de base de datos postgreSQL, hosteada en el servidor https://www.elephantsql.com/.

NodeJS

La conexión a la base de datos se realiza utilizando el entorno nodejs, el cual también se encarga de levantar al servidor.
